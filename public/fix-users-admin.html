🔧 Fix User Profiles - Admin Tool

⚠️ Admin Only: This tool updates existing user documents to include required fields for creating listings.
Please sign in as an admin to use this tool.

This will update all user documents that are missing the following required fields:

  • status - Set to 'approved'
  • isDisabled - Set to false
  • isBanned - Set to false
  • zipApproved - Set to false
  • blockedUsers - Set to []
  • displayName - Set from email
